# Tufte Inspired Theme

This is a barebones theme. It turns many aspects of plot background
lines, etc. off completely. Inspired from the excellent ggthemes
package: https://github.com/jrnold/ggthemes

## Usage

``` r
theme_tufte(base_size = 11, base_family = "sans")
```

## Arguments

- base_size:

  base font size, given in pts.

- base_family:

  base font family
